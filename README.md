# Embedded-Life-Search-and-Rescue-Detection-System-for-Tunnel-Collapse-and-Ruin-Scenarios
本项目基于国产沁恒 CH32V307（青稞 RISC-V 内核）芯片，研发面向坑道废墟封闭坍塌场景的嵌入式生命搜救探测终端。 系统采用 “振动 + 红外” 双源感知架构，全离线完成滤波去噪、包络提取、自适应阈值计算与敲击检测六级流水线处理；通过四级状态机识别规律敲击，5 秒内累计 3 次有效敲击即触发报警，辅以去抖机制抑制误判。 整机配备多级声光预警、3.5 寸 TFT 实时显示与串口调试功能，采用锂电池搭配 IP5306 充放电管理，便携离线续航超 6 小时。
